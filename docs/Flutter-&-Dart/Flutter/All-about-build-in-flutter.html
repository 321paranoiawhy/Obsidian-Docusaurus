flutter build appbundle

iOS ipa

# 切换至 ios 文件夹路径
cd ios

# 安装依赖
pod install

# 返回上一级目录 (根目录)
cd ..

# build
flutter build ipa

构建完成后, 在 XCode 中打开 ios 文件夹, 点击运行。运行完毕后双击 build/ios/archive 下的 Runner.xcarchive 文件 (用 XCode 打开), 然后一直下一步即可上传成功。

如果过程中报版本不一致的错误, 可进行如下操作:

cd ios
# 清除缓存
pod cache clean --all
# 删除 Podfile.lock 文件
rm Podfile.lock

cd ..
# 清除缓存
flutter clean
flutter pub get
# 更新
pod update
pod repo update

# 重新安装依赖
pod install

Web

检查是否支持 Web:

flutter devices

观察到 Chrome/Server 即表明支持 Web。

可通过如下命令开启 Web:

flutter config --enable-web

运行:

// 运行至 chrome 浏览器
flutter run -d chrome

打包:

// 渲染方式: auto / html / canvaskit
flutter build web --release
flutter build web --web-renderer html --release
flutter build web --web-renderer canvaskit --release

// 以 html 方式渲染、profile 模式运行至 chrome
flutter run -d chrome --web-renderer html --profile

// tree-shake
flutter build web --tree-shake-icons
提示

Flutter 的模式 (mode) 有:

  • debug mode
  • release mode
  • profile mode
