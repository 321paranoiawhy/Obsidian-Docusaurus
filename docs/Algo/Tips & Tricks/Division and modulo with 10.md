# Division(//) with (1, 10, 100,…)

- Num // (1,10,100,…) returns number including (1th,10th,100th) place and above.

```python
num = 123
print(num // 1)
print(num // 10)
print(num // 100)
```

# Modulo 10

-   Num % 10 return the last part of the number.

```python
num = 123
print(num & 10)
```
